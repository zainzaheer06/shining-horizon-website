Import Instructions

Import data from CSV files. Follow the order below for best results:

  1. Brands - Import first (no dependencies)
  2. Categories - Import second
  3. Subcategories - Import third (requires categories)
  4. Products - Import last (requires categories, subcategories, brands)

1. Import Brands

CSV columns: name, display_order

2. Import Categories

CSV columns: name, type, description, hero_title, hero_description, display_order

Type: simple or detailed

3. Import Subcategories

CSV columns: category_name, name, description, display_order

4. Import Products

CSV columns: category_name, subcategory_name, brand_name, name, part_number, short_description, description, display_order
